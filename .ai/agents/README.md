# AI Agent Configuration

## Purpose

- Provides agent-specific configuration and guidelines
- Tailors instructions for different AI coding platforms
- Optimizes agent performance for this specific project
- Maintains compatibility across agent versions

## Supported Agents

- **GitHub Copilot/Codex**: OpenAI-powered code completion
- **Claude Code**: Anthropic's AI coding assistant
- **Windsurf**: Multi-agent development platform
- **Future Agents**: Extensible framework for new AI tools

## Agent-Specific Guidelines

Each agent has specific capabilities and limitations:

- **Context Window**: Varies by agent (refer to agent-specific docs)
- **Language Support**: Different strengths per agent
- **Integration Methods**: API, CLI, or IDE-specific approaches
- **Customization**: Agent-specific configuration options

## Usage Instructions

1. Identify your AI agent platform
2. Review the corresponding agent-specific documentation
3. Apply the recommended configurations
4. Follow the platform-specific best practices

## AI Agent Guidelines

- Select the appropriate configuration for your platform
- Adapt general guidelines to your agent's capabilities
- Report agent-specific issues for continuous improvement
- Contribute back agent-specific optimizations
